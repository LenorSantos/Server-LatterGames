# Server-LatterGames
Backend do projeto Latter Games.

### Para executar:

#### Inicialmente:
```
npm i
```
Necessário para fazer a instação de todos os modulos.

#### Execução:
```
npm start
```

#### Execução em modo de desenvolvimento:

Instalar o nodemon global:
```
npm i -g nodemon
```

Depois:
```
npm run dev
```

##### Versão do node: *v22.12.0*
